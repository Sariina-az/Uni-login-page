# Login Page

Desined and developed by **Arian Seifpour**

## Project Overview

This is a test and practice project for presentation at the university. This project is a login page with appropriate and advanced styles, real-time validation, displaying errors and appropriate messages for users to enhance the user experience.

## Features

- Real-time validation
- Advanced styles
- Appropriate error messages with meaningful colors
- Display success messsage
- Preventing user password disclosure

## Installation

To install this project to your local computer, run these commands on your system:

```bash
    git clone https://github.com/Arisp5/login-page.git
    cd <project-folder>
```
Then to run project on your local system, use [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension. Right-click on the **HTML** file and select **Open with Live Server** or use the `Alt+L Alt+O` keyboard shortcut.
